# news.ieumedu.kr scaffold

이 폴더는 `ieumedu.or.kr` 교육원 사이트와 분리된 `news.ieumedu.kr` 별도 인터넷신문 사이트입니다.

현재 상태:

- 교육원 라이브 사이트와 분리된 정적 사이트
- 제호는 `이음교육저널`로 표기
- 등록번호, 등록연월일, 발행인, 편집인, 청소년보호책임자는 확정 전
- 별도 Pages 배포에서는 이 폴더를 사이트 루트로 사용

목적:

- 인터넷신문 등록용 공개 IA와 필수 공개 항목을 먼저 분리 설계
- 교육원 사이트와 다른 콘텐츠 성격을 명확히 보여 주기
- 별도 배포 대상(`news.ieumedu.kr`)으로 운영

현재 편집 방향:

- 기존 `think-sis` 블로그 아카이브를 검토해 기사화 가능한 원고를 선별
- 우선 축은 `생애주기와 수면`, `이론 브리프`, `특수·통합교육`
- 상세 목록은 `기사화-편성안.md` 참고
