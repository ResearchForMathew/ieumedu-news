# 블로그 원고 기반 기사화 편성안

## 기준

- 원고의 전문성을 유지하되, 기사 문법으로 다시 쓴다.
- 상담 사례는 개인 식별 정보와 구체적 사연을 그대로 노출하지 않는다.
- 교육원 홍보 문구가 아니라 독자가 읽을 정보·해설 중심으로 재작성한다.
- 초기 공개면은 해설, 기획, 현장, 이론 브리프 네 축으로 시작한다.

## 보류 원고

- `상담` 카테고리 원고는 정보성이 높지만, 개별 사례 묘사가 구체적이어서 바로 기사화하지 않는다.
- 기사화가 필요하면 사례를 2차 가공해 `익명화 + 유형화 + 전문가 코멘트` 구조로 다시 쓴다.
- `모래놀이심리상담사 2급 자격 인증`은 기사보다 인증 서비스/안내 페이지 성격이 강하므로 뉴스면에서 분리한다.

## 권장 첫 공개면

1. 대표 기획 1건: 영유아기 또는 청소년기 수면 기사
2. 해설 2건: 아동기 수면, 중년기 수면
3. 현장 1건: 발달지연 조기 발견
4. 이론 브리프 2건: 보울비, 반두라

## 칼럼 소스

- 브런치북 원고는 `브런치북-칼럼-검토.md` 기준으로 칼럼/에디터스 노트 축에 우선 배치한다.
- 특히 `잠은 최고의 보약이 아니다`, `회복은 일상에서 시작된다`, `프롤로그`는 칼럼 적합도가 높다.

## 편집 메모

- 제목은 블로그형보다 기사형으로 짧고 명확하게 줄인다.
- 첫 문단은 문제 제기와 핵심 요지를 바로 드러낸다.
- 문단 사이의 참고문헌은 기사형 본문에서는 줄이고, 끝에 `참고 자료`로 묶는다.
- 원고 안의 브런치·티스토리 내부 링크는 기사 공개면에서 직접 노출하지 않는다.
